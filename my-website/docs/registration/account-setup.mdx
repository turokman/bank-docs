# Настройка аккаунта
1. Настройте аккаунт. Для начала введите вашу почту.
2. Выберите тип учетной записи. После выбора **Далее**.
3. Добавьте адрес регистрации (адрес, город или регион, почтовый индекс). Нажмите **Далее**.
4. Добавьте личные данные (имя и дата рождения). Нажмите **Далее**.
5. Добавьте свою карту, впишите необходимые данные. Нажмите **Далее**.
6. Дождитесь, пока на ваш номер телефона придет 6-значный код. Введите полученный код и подтвердите номер телефона.
7. Вы должны увидеть уведомление "Карта успешно привязана", а также в разделе **Мои карты** будут указаны последние четыре цифры вашей карты.
8. Поздравляем, регистрация завершена. Нажмите **Продолжить**, чтобы начать пользоваться приложением.
